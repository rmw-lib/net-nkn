# @rmw/net-nkn

请求文件 时间戳 签名 路径列表 
响应 时间戳 签名 路径 文件 


更新 dns

每隔1秒请求下一个

一旦响应 就回调

检查回调是否还存在

如果请求结束还没有

等待30秒



# 请求推送

标识 公钥 路径 

# 响应

标识 公钥 签名 路径 文件内容  


##  安装

```
npm install @rmw/net-nkn
```

## 使用

```
import net-nkn from '@rmw/net-nkn'

```
